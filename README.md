| INSTALL BOT |
|:-----------------------|
- <p align="left">Clone Source:
```sh
git clone https://github.com/robosaz/xtfa && cd xtfa
```
- <p align="left">install bot:
```sh
chmod 700 start.sh
./start.sh install
y
```
| ANTI CRASH |
|:-----------------------|
```sh
tmux new-session -s script "bash steady.sh -t"
```

`./start.sh -q`

| FREE SERVER: |
|:-----------------------|
